# insurance-policy-sql
Project that implements SQL and Python for an insurance policy management system.

## How to Run

### 1. Install dependencies
```bash
pip install flask oracledb
```

### 2. Connect to Penn State network
The database is hosted at Penn State. If you are off-campus, connect to the **GlobalProtect VPN** before running the app.

### 3. Run the app
```bash
python app.py
```

### 4. Open in browser
Go to:
```
http://127.0.0.1:5000
```

To stop the app, press `Ctrl + C` in the terminal.
